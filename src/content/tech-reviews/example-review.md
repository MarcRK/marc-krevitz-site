This is an example review, not a real one — it exists so you (Marc) can see how a review post is structured before writing your first real one.

## How to add a real review

1. Copy this file into `src/content/tech-reviews/` and rename it, e.g. `moondrop-blessing3-review.md`.
2. Fill in the front-matter:
   - `title` — the review title.
   - `date` — publish date (`YYYY-MM-DD`).
   - `product` — the product name.
   - `summary` — one or two sentences for the list card.
   - `draft` — set to `false` (or delete the line) once it's ready to publish.
3. Write the review body below — your honest take.
