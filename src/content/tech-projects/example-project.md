This is an example entry, not a real project. It's here so I can see how a Tech Projects post is structured before writing the first real one.

## How to add a real entry

1. Copy this file into `src/content/tech-projects/` and rename it, e.g. `gameboy-sp-refurb.md`.
2. Fill in the front-matter. `title` is the project title. `date` is the publish date (`YYYY-MM-DD`). `category` is something like "Refurbishment" or "Audio DIY". `summary` is one or two sentences for the project list card. `draft` should be `false` (or removed) once it's ready to publish.
3. Add `images` (before/after or build photos) once saved in `public/images/tech-projects/`, same format as the furniture collection.
4. Write the body using the log format: what machine or build it was, what was wrong or planned, what got fixed or built, and the outcome.

Headphone and IEM reviews are a separate collection, see `src/content/tech-reviews/`.
