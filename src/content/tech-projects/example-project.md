This is an example entry, not a real project — it exists so you (Marc) can see how a Tech Projects post is structured before writing your first real one.

## How to add a real entry

1. Copy this file into `src/content/tech-projects/` and rename it, e.g. `gameboy-sp-refurb.md`.
2. Fill in the front-matter:
   - `title` — the project title.
   - `date` — publish date (`YYYY-MM-DD`).
   - `category` — e.g. "Refurbishment", "Audio DIY".
   - `summary` — one or two sentences for the project list card.
   - `draft` — set to `false` (or delete the line) once it's ready to publish.
3. Optionally add `images` (before/after or build photos) once saved in `public/images/tech-projects/`, same format as the furniture collection.
4. Write the body using the log format: what machine/build it was, what was wrong or planned, what you fixed or built, and the outcome.

Note: headphone and IEM reviews are a separate collection — see `src/content/tech-reviews/`.
